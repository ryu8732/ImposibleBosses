# Impossible Bosses

## 프로젝트 소개

Unity 엔진을 기반으로 제작된 MO 액션게임으로, 워크래프트3의 유즈맵 "불가능한보스잡기"를 참고하여 제작하였습니다.


## 프로젝트 개요

- 개발기간: 4주

- 개발도구: Unity

- 개발언어: C#

- 플랫폼 : 안드로이드
